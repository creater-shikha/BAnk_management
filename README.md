# BAnk_management
A bank management system using C++ contain all basic information that  is required for any bank...
System show following detail
  >>Account number
  >>Account holder name(first & last)
  >>Balance
Facilities provide
  >>Open account
  >>close account
  >>Deposite amount
  >>Withdrawal
  >>Balance Enqurie
  >>show all account
  >>quit

     >>iostream

   >>file Handling 
   >>exception
   >>overloading 
   >>constructor
